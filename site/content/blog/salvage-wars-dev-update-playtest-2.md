***Salvage Wars:*** a tactical RTS / top-down extraction shooter hybrid where your mercenary crew fights to salvage wreckage from an orbital war.

This build marks **Playtest 2** and introduces the first version of the overworld + salvage loop that ties missions together.

The focus of the playtest is still combat feel, but now operations have real stakes since you can carry losses and profits between deployments.

### What's New

- **Overworld progression loop** connecting multiple operations
- **Randomized encounters** with selectable difficulty
- **One-click salvage orders**
- **Salvaging UI** for identifying and transferring recovered components
- **Shop system** to sell loot and purchase equipment
- **Unit loadout editor** for customizing vehicles and crews

### Current State

The overworld is still very early and mostly scaffolding. Many visuals are placeholder and the UI will continue to evolve as systems solidify.

The goal of this milestone was simply to get the ore loop working, so combat decisions have consequences: deciding when to push for more salvage, when to extract, and how to manage losses between operations.

### Playtesting

This is still a **private playtest**. If you'd like to try the current build, feel free to reach out to me to be added.
